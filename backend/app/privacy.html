IG Content Coach 개인정보 처리방침

본 GPT는 인스타그램 계정 데이터 중 사용자가 직접 입력한 정보(예: URL, 게시물 통계)만 분석에 사용하며, OpenAI API를 통해 모델 응답을 생성합니다. 사용자의 데이터는 서버에 저장되지 않으며, 분석 결과 생성 후 즉시 폐기됩니다.

1. 수집하는 정보

  • 사용자가 직접 입력한 계정 URL 및 게시물 통계 데이터
  • 자동 수집 정보 없음 (쿠키, IP 로그 저장 없음)

2. 이용 목적

인스타그램 콘텐츠 성과 분석 및 개선 피드백 제공을 위해서만 사용됩니다.

3. 보유 및 이용 기간

입력 데이터는 서버 메모리에서만 처리되며, 저장 또는 로그로 남지 않습니다.
